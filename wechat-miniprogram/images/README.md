# 图标文件说明

此目录包含微信小程序的标签栏图标文件。

## 需要的图标文件：

### 连接标签
- `tab-connect.png` - 连接图标（未选中状态）
- `tab-connect-active.png` - 连接图标（选中状态）

### 文件传输标签  
- `tab-file.png` - 文件图标（未选中状态）
- `tab-file-active.png` - 文件图标（选中状态）

### 设置标签
- `tab-settings.png` - 设置图标（未选中状态）
- `tab-settings-active.png` - 设置图标（选中状态）

## 图标规格要求：
- 尺寸：81px × 81px
- 格式：PNG
- 背景：透明
- 颜色：未选中状态使用灰色 (#666666)，选中状态使用主题色 (#2196F3)

## 临时解决方案：
在开发阶段，可以使用以下方式：
1. 使用在线图标生成器创建简单图标
2. 使用 emoji 字符作为临时图标
3. 从开源图标库下载合适的图标

## 图标来源建议：
- Feather Icons
- Material Design Icons  
- Heroicons
- Tabler Icons
